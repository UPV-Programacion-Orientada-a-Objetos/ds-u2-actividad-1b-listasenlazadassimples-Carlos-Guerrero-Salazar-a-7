Detailed Description

Lista enlazada NO genérica para gestión polimórfica de sensores.

Almacena punteros a SensorBase* permitiendo el polimorfismo. Es responsable de liberar la memoria de los objetos de sensor apuntados al ser destruida (propiedad de la lista).
