Detailed Description

template<typename T>
class ListaSensor< T >

Lista enlazada genérica simple para almacenar datos de tipo T.

  • Esta clase implementa una lista enlazada simple, diseñada para ser utilizada dentro de clases específicas de sensores para manejar sus lecturas o registros.
  • Template Parameters
    TEl tipo de dato a almacenar en la lista (ej., int, float, o una estructura).
